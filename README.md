# 🚨 SafeSignal – Smart Crisis Response System

## 🔍 Problem

In emergency situations (accidents, unsafe areas, medical issues), people often **fail to get timely help** due to:

* Lack of real-time alerts
* No nearby assistance visibility
* Delayed communication

---

## 💡 Solution

**SafeSignal** is a real-time crisis response platform that:

* Detects and reports incidents
* Shows nearby help/resources
* Sends SOS alerts instantly
* Provides live location tracking

---

## ⚙️ Features

* 🚨 SOS Emergency Alert
* 📍 Live Location Sharing
* 🗺️ Nearby Help Detection
* 💬 Real-time Chat
* 🔔 Instant Notifications
* 🌐 Interactive Map (OpenStreetMap)

---

## 🛠️ Tech Stack

**Frontend:**

* React.js
* Tailwind CSS
* Leaflet.js

**Backend:**

* Node.js
* Express.js

**Database & Services:**

* Supabase (PostgreSQL + Auth)

## 🧠 How It Works

1. User triggers SOS
2. Location is captured
3. Data sent to backend
4. Nearby users/services notified
5. Help reaches faster
